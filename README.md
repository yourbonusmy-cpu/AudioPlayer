# 🎵 Python Audio Player (PyQt6 + SQLAlchemy)

Современный локальный аудиоплеер, разработанный на Python с использованием **PyQt6**, **SQLAlchemy (SQLite)** и визуализацией аудио в виде waveform.

## 🚀 Основные возможности

### 🎧 Аудиоплеер
- Воспроизведение MP3 файлов
- Play / Pause
- Переключение треков (⏮ ⏭)
- Автоматический переход к следующему треку
- Запоминание позиции воспроизведения
- Продолжение с места остановки после перезапуска

---

### 📊 Waveform (визуализация аудио)
- Реальный waveform трека
- Отображение прогресса воспроизведения
- Перемотка по клику на waveform
- Кеширование waveform в SQLite (ускорение загрузки)

---

### 📁 Работа с файлами
- Выбор музыкальной папки
- Поддержка Linux / Windows
- Рекурсивное сканирование папки
- Drag & Drop папки в окно приложения
- Обновление списка треков

---

### 🏷 Теги
- Создание пользовательских тегов (например: `rock`, `спорт`, `грусть`)
- Привязка тегов к трекам через checkbox
- Фильтрация по тегам
- Удаление тегов с подтверждением (ввод имени тега)
- Сохранение связей тегов в базе данных

---

### 🔍 Поиск и фильтрация
- Поиск по названию трека
- Фильтр по тегам
- Debounce поиск (500ms задержка)

---

### 💾 База данных (SQLite + SQLAlchemy)

Хранится:
- Треки
- Путь к музыкальной папке
- Теги и связи
- Waveform (кэш)
- Последняя позиция трека
- Количество проигрываний
- Настройки (громкость, папка)

---

### 🔊 Управление звуком
- Слайдер громкости
- Сохранение громкости между запусками

---

## ⚙️ Технологии

- Python 3.10+
- PyQt6
- SQLAlchemy
- SQLite
- Mutagen (метаданные аудио)
- NumPy (waveform)

---

## 📌 Особенности

- Полностью оффлайн приложение
- Быстрый старт даже с большими библиотеками
- Кэширование waveform для ускорения UI
- Сохранение состояния между запусками
- Минималистичный UI в стиле современных музыкальных сервисов
